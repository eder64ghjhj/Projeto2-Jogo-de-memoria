
# 🧠 Jogo da Memória

Jogo da memória interativo desenvolvido com HTML5, CSS3 e JavaScript puro. O jogador vira cartas tentando encontrar os pares correspondentes.

## 🎮 Como Jogar

1. Clique em uma carta para virá-la
2. Clique em outra carta tentando encontrar o par
3. Se as cartas forem iguais, elas permanecem viradas
4. O objetivo é encontrar todos os pares no menor número de tentativas

## ✨ Funcionalidades

- Cartas embaralhadas aleatoriamente a cada partida
- Detecção automática de pares
- Contagem de tentativas
- Animação de virada das cartas

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
